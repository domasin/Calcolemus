allpairs f l1 l2

Full Usage: allpairs f l1 l2

Parameters:
    f : 'b -> 'c -> 'd
    l1 : 'b list
    l2 : 'c list

Returns: 'd list

Compute list of all results from applying function to pairs from two lists. The call allpairs f [x1;...;xm] [y1;...;yn] returns the list of results [f x1 y1; f x1 y2; ... ; f x1 yn; fx2 y1 ; f x2 y2 ...; f xm y1; f xm y2 ...; f xm yn ] Never fails.

f : 'b -> 'c -> 'd
l1 : 'b list
l2 : 'c list
Returns: 'd list
