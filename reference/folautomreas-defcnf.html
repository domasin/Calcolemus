A triple with the transformed formula, the augmented definitions and a new variable index.

Example 

 maincnf (!> @"p \/ (p \/ q)", undefined, 0I)
Evaluates to: 
 (`p_1`,
   `p 
   `p 
   2I)

n : bigint

Returns: formula<prop> * bigint

The indexed variable and the next index

Example 

 mkprop 3I
Evaluates to 
 (`p_3`, 4 {IsEven = true;
            IsOne = false;
            IsPowerOfTwo = true;
            IsZero = false;
            Sign = 1;})
