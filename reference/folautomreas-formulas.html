parse_formula (ifn, afn) vs inp

Full Usage: parse_formula (ifn, afn) vs inp

Parameters:
    ifn : string list -> string list -> formula<'a> * string list
    afn : string list -> string list -> formula<'a> * string list
    vs : string list
    inp : string list

Returns: formula<'a> * string list

Recursive descent parser of polymorphic formulas built up from an atomic formula parser by cascading instances of parse infix in order of precedence, following the conventions with '/\' coming highest and '<=>' lowest. It takes a list of string tokens `inp` and In order to check whether a name is within the scope of a quantiﬁer, it takes an additional argument `vs` which is the set of bound variables in the current scope. It returns a pair consisting of the parsed formula tree together with any unparsed input.

ifn : string list -> string list -> formula<'a> * string list
afn : string list -> string list -> formula<'a> * string list
vs : string list
inp : string list
Returns: formula<'a> * string list
