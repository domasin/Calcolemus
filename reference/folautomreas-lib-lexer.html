Takes a property prop of characters, such as one of the classifying predicates (Lexer.space, Lexer.punctuation, Lexer.symbolic, Lexer.numeric, Lexer.alphanumeric), and a list of input characters inp, separating off as a string the longest initial sequence of that list of characters satisfying prop.

Generic function to impose lexing and exhaustion checking on a parser.

A wrapper function that explodes the input string, lexically analyzes it, parses the sequence of tokens and then internally checks that no input remains unparsed.
